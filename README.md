# ShopSmart
ShopSmart – MERN E-commerce Web Application
Project Description
ShopSmart is a full-stack e-commerce web application built using the MERN stack. Users can
browse products, add items to cart, register/login securely, and place orders. Admin users can
manage products, users, and orders through a dashboard. This project demonstrates full-stack
development including authentication, REST APIs, and database integration.
Tech Stack
• Frontend: React.js, Vite
• Backend: Node.js, Express.js
• Database: MongoDB Atlas
• Authentication: JWT
• Version Control: Git & GitHub
Features
• User Registration & Login
• Browse Products
• Add to Cart
• Place Orders
• Admin Dashboard
• Product Management (Add / Edit / Delete)
• Protected Routes using JWT
Installation & Setup
1. Clone the repository:
git clone https://github.com/YOUR-USERNAME/ShopSmart.git

3. Setup Backend:
cd backend
npm install
npm start

5. Setup Frontend:
cd frontend
npm install
npm run dev
Environment Variables (.env in backend)
PORT=5100
MONGO_URI=your_mongodb_atlas_connection_string
JWT_SECRET=your_secret_key
